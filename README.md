# eromatome-dl

Downloader for まとめ blogs.

Supports `moeimg.net`, `pashalism.com`, `kimootoko.net`,
`ichinuke.com`, `erocon.gger.jp`, `nijifan.net`, `beppin-girl.com`,
`ero-gazou.jp`, `bakufu.jp`, `1000giribest.com`, `les-koko.com`,
`hnalady.com`, `yaruo.info`, `erogazou.gallery`, and `erokan.net` and many other matome sites.

## Run

Grab the .exe from Releases.  

Python 3.10+ is required. Tkinter is part of the standard Python installers for
Windows and macOS. On some Linux distributions it is packaged separately.

```bash
python -m eromatome_dl
```
